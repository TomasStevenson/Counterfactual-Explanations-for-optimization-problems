# Counterfactual Explanations for Optimization Problems

This repository contains implementations and experiments for counterfactual explanations in optimization problems.

## Project Structure

- `RelativeCE.ipynb`: Main Jupyter notebook with the implementation.

## Dependencies

Install the required packages using:

```bash
pip install -r requirements.txt
```

### Gurobi License

This project uses Gurobi optimizer, which requires a valid license. Make sure to set the `GRB_LICENSE_FILE` environment variable to point to your license file, as shown in the notebook.

## Usage

Open and run the `RelativeCE.ipynb` notebook in Jupyter.

## Contributing

Please refer to the notebook for detailed code and explanations.
